# heli_WebGL

![](resources/heli.gif)

WebGL2를 이용해 헬리콥터 구현

## 시작하기

https://webglreport.com/?v=2 를 접속하여 브라우저의 WebGL2 지원 여부를 확인합니다.

vscode의 liveserver를 이용하여 실행합니다.

## 조작 방법
### 헬기 이동
* `←`: 왼쪽 회전
* `→`: 오른쪽 회
* `↑`: 앞으로 이동
* `↓`: 뒤로 이동
* `A` or `a`: 위로 이동
* `Z` or `z`: 아래로 이동
### 카메라 이동
* `Shift`+`←`/`Shift`+`→`: 카메라 회전
* `Shift`+`↑`/`Shift`+`↓`: 틸트
* `=` or `+`: 줌 인
* `-` or `_`: 줌 아웃
